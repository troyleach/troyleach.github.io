JavaScript, Pros and Cons

August 2, 2014

Some Pros


Easy to learn

JavaScript is easy to learn because of the very flexible nature of the language. It is now widely standardized across common browsers. Because JavaScript is not a compiled language, things like memory management are not a big concern.



Easy to edit

JavaScript is easy to get started with because you don't need much to do so. JavaScript is a scripting programming language so the code does not need to be compiled. JavaScript is just plain text, so all you need to write it is a plain text editor.



Allows for object oriented design

Obviously there are other OOP languages that come to mind before JavaScript, however it is able to make use of object oriented design patters and principles. Unlike more structured languages, however, JavaScript does not force you to use OOP.




Some Cons


JavaScript cannot write to files on the server without the help of a server side script.

JavaScript can not write to files on the server without the help of a server side script. Using Ajax, JavaScript can send a request to the sever, this request can read a file in plain text, but it can not write to a file unless the file called on actually runs as a script.



Security

Some say that using JavaScript has security issues. ONE of the concerns from folks is JavaScript snippets, once appended onto web pages execute immediately and in some cases it can be exploited for malicious purposes, such as delivering viruses and adding advertising programs onto your computer. This leads me into my next con.



DON'T disable

Because of the above issue, some users decide to disable JavaScript. JavaScript is basically complied and run by web browsers. If the user decides to disable JavaScript some things on your site might not work.




In conclusion


From the research I did, developers are becoming more and more sophisticated in how they think about the language. While a few years ago most people thought JavaScript as useful for controlling the presentation of a page, developers are starting to think of JavaScript as more full-featured language.

JavaScript is quickly becoming not just the ideal language for the web, but indeed a big composition of the web itself. The future of the web looks bright, and more importantly free for developers everywhere.
