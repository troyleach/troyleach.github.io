GIT Commands

Create Repositories

git init [project-name]

Creates a new local repository with the specified name

git close [url]

Downloads project and its entire version history

Make Changes

git status

Lists all new or modified files to be committed

git add [file]

Snapshots of the file

git commit -m "[descriptive message]"

Description of the changes to the file

git push origin master

Pushes the files to github

Group changes

git branch

Displays the current branch, can also list all branches.

git branch [branch-name]

Creates a new branch

git checkout [branch-name

Switches to the specified branch and updates the working directory

git merge [branch]

Combines the specified branch's into the current branch

git branch -d [branch-name]

Deletes the specified branch

Useful commands

git diff HEAD

check to see what was changed

git diff --staged

See what is staged

git reset "[file-name"]

Remove a file that is staged
